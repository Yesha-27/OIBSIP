# Oasis Infobyte Java Development Internship (OIBSIP)

This repository contains all the tasks completed during my Java Development Internship at Oasis Infobyte.

## Completed Tasks

### Task 1 - Online Reservation System
A Java-based railway reservation system with login, reservation, and ticket cancellation using PNR.

### Task 2 - Number Guessing Game
A Java console application where users guess a randomly generated number.

### Task 3 - ATM Interface
A Java application simulating ATM operations such as deposit, withdrawal, transfer, and transaction history.

### Task 4 - Online Examination System
A Java console-based examination system with login, MCQ test, timer, and automatic result generation.

### Task 5 - Digital Library Management System
A Java application for managing books including add, search, issue, return, and remove operations.
